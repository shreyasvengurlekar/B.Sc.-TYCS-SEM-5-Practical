Run:

FLUSH PRIVILEGES;
EXIT;


Replace 'your_password' with your actual root password.

‚úÖ 9. Secure phpMyAdmin (Optional but Recommended)
üîí Use .htaccess (basic auth)
sudo nano /etc/phpmyadmin/apache.conf


Add this line inside the <Directory /usr/share/phpmyadmin> block:

AllowOverride All


Then create .htaccess:

sudo nano /usr/share/phpmyadmin/.htaccess


Paste:

AuthType Basic
AuthName "Restricted Access"
AuthUserFile /etc/phpmyadmin/.htpasswd
Require valid-user


Create a user and password for login:

sudo apt install apache2-utils
sudo htpasswd -c /etc/phpmyadmin/.htpasswd youruser


Restart Apache:

sudo systemctl restart apache2
